# Warehouse Operations Analysis

## Overview
Analysis of 100,000+ real e-commerce orders from the Brazilian Olist dataset to identify delivery performance patterns across regions and product categories.

## Tools Used
- SQL (DB Browser for SQLite)
- Excel

## Business Questions Answered

### 1. Which product categories have the highest late delivery rate?
- casa_conforto_2 had the highest late rate at 16.67%
- Audio products had 362 orders with 12.71% late delivery rate

### 2. Which regions have the slowest delivery times?
- RR was slowest at 29.39 average days
- SP was fastest at 8.76 average days

### 3. Do high value orders get delivered faster or slower?
- Low value orders (<50): 11.38 days average
- Mid value orders (50-200): 12.86 days average  
- High value orders (>200): 13.94 days average
- Finding: Higher value orders take longer to deliver
